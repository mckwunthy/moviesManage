################################### APPLICATION DE GESTION DE FILM #############################################
--->Description : gerer une collection (base de données) de films
			avec les fonctionnalités de : --> ajout
								--> recherche & filtrage
								--> modification & suppression

---> Interface : deux blocs : gauche et droit
		--> bloc gauche : --> barre de recherche rapide
							faire des recherches rapide à partir de : titre, actors, annee, realisateur
					-->box d'affichage : affiche les informations de la vidéo selectionnée
		
		--> bloc droit :  -->boutton d'ajout (add movie)
							ajouter un nouveau film dans la base de donnée
			 		-->box base de donnée : 
							reparrtorie toutes les videos enregistrées
					
					--> boutton view : voir les informations d'une video
					--> boutton updat : mettre à jour les informations d'une vidéos
					-->button delete : supprimer une video


NB : - usage sur ecran de pc
	 - appli non responsive
